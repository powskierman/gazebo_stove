# Gazebo Stove Project

## Overview
The **Gazebo Stove** project is an ESPHome-based heating control system for an outdoor gazebo. It replaces a legacy Arduino/Blynk implementation with a local-first, Home Assistant-integrated solution.

**Current Status**: Implementation Phase (Phase 2 Complete)
